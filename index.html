Lecture 1

This lecture dicusses the main data formats that are used for sharing and transfering information over the Web. These data formats are the backbone of the services provided, and used, on the Web. Make sure to get the basics of them, it will help you to get the grip on this course. Following is the list of data formats that will be discussed.
  • XML, DTD, XSD

    • The Web is full of XML documentation. However, I quite like the way it is presented on UC Berkeley Website (Link).

  • JSON

    • The official website (Json) contains some information about the syntax and usage. Other sources include W3schools, Scot's Blog, etc.

  • RDF, OWL

    • I would recomend every one to read -- at-least -- first 40 pages of Linked Data and the Semantic Web Standards by Aidan Hogan (can be downloaded at this Link). It will provide the basics of the Semantic Web, RDF, OWL and Linked Data.
