Middleware Architecture

Course Objectives

"Middleware is a software that runs between client and server processes. It is the "glue" between the client and server, which makes it possible for them to communicate to each other. Middleware is written in such a way that the user never notices it's presence." The objective of this course and the labs are to provide introduction about how to build the middleware architecture using JAVA EJB. Note that there are also various other technologies for such task, howevere JAVA EJB are extensively used in the industry.

The main instructor of this course is Julien Subercaze (julien [dot] subercaze [at] univ-st-etienne [dot] fr), therefore for further details please contact him directly.
